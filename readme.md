# Coffee Delivery Project

This application is a challenge made by rocketseat for students

## ⚙️ Installation

Install Dependencies

```bash
  npm install
```

Run Project

```bash
  npm run dev
```

Build Project

```bash
  npm build
```

## 🖥️ Technologies Used

- React (18.2.0)
- Typescript (4.9.3)

## 📚 Libraries Used

- radix-ui (Radio Group)
- react-input-mask
- react-hook-form
- yup
- react-router-dom
- styled-components
- phosphor-react

## 🚀 Features

- Responsiveness
- State Manipulation
- Route Information Passing
- Context API
- Design with Styled Components
- Components Creation
- Form Creation
- Form Validation
- Typing
- Utilization Of Outlet for Header
